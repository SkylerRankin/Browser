HTML Web Browser

This browser implements the bare minimum HTML and CSS required to view simple webpages. The rendering pipeline converts a given URL into a visible webpage via three main steps: resource fetching, layout calculation, and rendering. These steps, particularly layout calculation, explore lots of interesting HTML rendering engine problems, such as DOM box calculation, resolving messy CSS precedence rules, and intelligent line splitting.

Open/close the developer tools with Ctrl + i.

Read more and see the source code at github.com/SkylerRankin/browser.
