kelp
~ a rendering engine and web browser in Java

Kelp is an HTML and CSS rendering engine written in Java with a minimal browser UI built with JavaFX.

Much of the core HTML 5 and CSS 3 specification required for static webpages is implemented. This includes standard features such as flow layout, lists, tables, various selectors, media queries, etc. While clickable links are supported, other interactive features that require repainting such as hover CSS effects, forms, and scripting in general are not yet implemented.
